## 基础知识

### 2.1 操作最右边的位元

```c++
//将最靠右值为1的bit修改为0
//00011010操作后变为(00011010&00011001)=00011000
//可以用来判断x是否2的幂次方（包含0），比如8为100，操作后为0
int i = x & (x-1);

//将尾部的值为1的bit变成0，比如1010 0111操作后变为(1010 0111&1010 1000)=1010 0000
//可以用来判断x是否为为2^n-1或者0，只要i=0
int i=x&(x+1);

//将值为0且最靠右的bit变为1，如果不存在值为0的bit，那么所有的bit都是1
//比如1010 0111，x+1为1010 1000，结果为1010 1000
int i=x | (x+1);

//将尾部的bit由1变为0  
//比如1010 0111，x-1为1010 0110，结果为1010 0111
int i=x|(x-1);

//将最靠右且值为0的bit变为1，其余bit变为0
//比如1010 0111，~x为0101 1000, x+1为1010 1000 ,结果为0000 1000
int i=~x&(x+1);

//将x中最靠右的值为1的bit变为0，并将其余bit变为1
//比如1010 1000，~x为0101 0111，x-1为1010 0111，结果为1111 0111
int i=~x|(x-1);
```
